# Typography

###### Usage

-   Accents
-   Logos

| Symbol                             | Value                         |
| ---------------------------------- | ----------------------------- |
| `S.TYPOGRAPHY_TEXT_FONT`           | 'Source Sans Pro', sans-serif |

| `S.TYPOGRAPHY_BASELINE`            | `SCALE_STEP` \* 2             |
| `S.TYPOGRAPHY_FONT_RATIO`          | 0.875                         |
